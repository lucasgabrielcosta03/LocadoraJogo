###
1- Requisitos para rodar o codigo em PHP ter alguma tipo de emulador de servidor exemplo (Xampp etc)
2- Colocar a pasta locadora dentro da pasta "htdocs" para garantir que seja visto a pagina pelo navegador 
3- Estartar o xampp com o Apache e o Mysql 
4- Acesssar colocar na URL do seu navegador o caminho da para a pasta que sera ( http://localhost/locadora/locadora/locadoraJogo/ ) 

##
Se quiser ver a incerção dos dados no banco de dados abra o "phpMyAdmin" se nao souber como abrir segue o passo a passo 

1- digite na sua URL "localhost" com o Apache e Mysql iniciados no Xampp 
2- va ate phpMyAdmin 
3- Clique em locadorajogo
4- Clique na sua tabela de preferincia Ex: tblcomprador
